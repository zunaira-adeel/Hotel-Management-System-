# Hotel Management System in C

## Abstract

The Hotel Management System is a simple console-based application developed using the C programming language. The system enables users to perform basic hotel operations such as room registration and food ordering through a menu-driven interface. This project demonstrates fundamental programming concepts including functions, loops, conditional statements, and user input handling while providing a practical real-world application of C programming.

---

# Introduction

## Purpose

The purpose of this project is to automate basic hotel management operations by replacing manual procedures with an interactive computerized system. The program focuses on simplicity, efficiency, and user interaction using foundational C programming concepts.

The system allows hotel staff or users to:

* Register and allocate rooms
* Manage guest details
* Order food from a hotel menu
* Generate bills for services

# Proposed Model

The proposed system is divided into two main modules that represent essential hotel operations.

---

## Module Breakdown

### Room Registration Module

This module:

* Collects guest information
* Registers customers
* Assigns available rooms based on user input
* Displays room details

---

### Hotel Menu Module

This module:

* Displays available food items
* Allows guests to place food orders
* Calculates food bills
* Generates the final payable amount

# Challenges Faced During Development

## No Persistent Storage

The system does not permanently store data because database integration is not included.

---

## Limited Error Handling

Only basic validation is implemented, so incorrect user inputs may affect program execution.

---

## No Graphical Interface

The project uses a command-line interface instead of a graphical user interface, limiting visual interaction.
